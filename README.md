# Template for C++ programs on profanOS

A little template for my C++ programs on [profanOS](<https://github.com/elydre/profanOS>) :D
It does not have the std headers. (I'm gonna wait for them to come on profan, I am too lazy to build it on my own)
But you can still use the OOP concepts !

## Compiling the program

```
mkdir build && cd build
cmake ..
make
```

And then you got your executable file in your build directory, ready to be ran on profanOS !
